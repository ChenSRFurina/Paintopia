# 概述
本项目是一款基于 Swift（SwiftUI）开发的创新型多端绘画与绘本生成软件，结合 LLM（大语言模型）和 VLM（视觉语言模型）能力，帮助用户在创作过程中获得智能建议，并能将绘画内容自动生成动画和绘本，提升创作乐趣与效率。目标用户为有创作需求的儿童、家长、教育者及绘画爱好者。

# 核心功能
1. 首页（绘画页面）
   - 左侧：工具栏（初期可为空或仅含基础工具）
   - 中间：大画板（Canvas），支持自由绘画
   - 右侧：LLM 智能建议区，实时展示建议内容
   - 画布下方：生成按钮
   - 智能建议：定时截取画布内容，VLM 画面理解，LLM 生成建议
   - 生成按钮：跳转生成页
2. 生成页面（动画与绘本）
   - 动画生成：根据绘画内容自动生成动画
   - 绘本生成：自动生成配文绘本，支持翻页、查看、下载（PDF/图片集）
3. 用户流程
   - 进入首页绘画 → 获得智能建议 → 点击生成 → 自动生成动画/绘本 → 浏览/下载

# 用户体验
- 主要面向 iPad 优先，兼容 Mac（Catalyst）
- UI 需简洁、易用，适合儿童和家庭用户
- 支持响应式布局，适配多端
- 画板操作流畅，建议区信息实时更新
- 生成页支持动画播放与绘本翻页

# 技术架构
- 前端：Swift（SwiftUI），支持 iOS 15+、macOS 12+（Catalyst）
- 画板：自定义 Canvas，支持基础绘画（画笔、橡皮、撤销等，初期可简化）
- 智能建议：通过 API 调用后端 LLM/VLM 服务
- 动画/绘本生成：本地或云端服务，API 对接
- 文件结构建议：Views/、Models/、Services/、Utils/

# 开发路线图
## MVP 阶段
- 首页（绘画页）基础搭建
- 画板基本功能实现
- 智能建议区 UI 与 API 对接（可用 mock 数据）
- 生成页基础搭建，支持动画/绘本展示（初期可用占位内容）
- 支持 iPad、Mac Catalyst 适配
- 导出/下载功能初步实现

## 后续扩展
- 丰富绘画工具与图层管理
- 用户账号与作品保存

# 逻辑开发顺序

- 动画/绘本生成效果：MVP 阶段可用占位内容，逐步完善
- UI/UX 适配儿童与家庭用户：多轮测试与反馈
